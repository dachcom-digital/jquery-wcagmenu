1. Use tab key, to focus menu.
  2. Use left and right cursor key for first level of navigation (loop). Use down key for opening second level.
  3. Use left and right cursor key for second level of navigation. Use down key for opening third level. Use up key for closing second level
  4. Use up and down cursor key for third level of navigation. Use right key for opening fourth level. Use up key on first item for closing third level
  5. Use up and down cursor key for fourth level of navigation (loop). Use left key for closing fourth level.
  6. Use esc any time to close the menu.
  7. Use tab key on focused menu to close menu and lose focus.
  8. Use space key to open submenu, when present. Otherwise space opens links.
  9. Use enter key to open links directly.
